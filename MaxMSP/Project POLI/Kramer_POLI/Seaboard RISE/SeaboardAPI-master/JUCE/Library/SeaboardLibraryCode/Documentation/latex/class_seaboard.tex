\subsection{Detailed Description}
\hyperlink{class_seaboard}{Seaboard} Class connects to the \hyperlink{class_seaboard}{Seaboard} as a M\-I\-D\-I\-Device, and converts incoming messages into a series of \hyperlink{class_seaboard_1_1_listener}{Seaboard\-::\-Listener} callbacks

To use, subclass Seboard\-::\-Listener, and implement the virutal methods relevant to you. You then need to register your class to the \hyperlink{class_seaboard}{Seaboard} objects Listener\-List using the \hyperlink{class_seaboard_a42baffad758f8eda105157b56234c668}{add\-Listener()} method
