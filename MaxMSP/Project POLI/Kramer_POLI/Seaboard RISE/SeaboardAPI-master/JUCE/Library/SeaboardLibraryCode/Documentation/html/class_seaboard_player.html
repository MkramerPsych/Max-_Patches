Detailed Description

The SeaboardPlayer is used to play sounds in response to incoming data from the seaboard. In order to produce sound from seaboard input you should create a SeaboardPlayer and add it as a listener to a Seaboad object (using the addListener() function).

In order to play sound you can then add custom voices using addNewVoice(). These custom voices should inherit SeaboardVoice.
